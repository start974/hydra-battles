First_toggle
Computes the first l between n and p (excluded) such that P l = true and P (S l) = false.
Pierre Castéran, Univ. Bordeaux and LaBRI

From Coq Require Import Arith Lia.

Section Hypos.
  Variables (P : nat → bool)
            (n p : nat).

  Hypotheses (Hnp : n < p)
             (Hn : P n = true) (Hp : P p = false).


  Definition first_toggle :
  {l : nat | n ≤ l ∧ l < p ∧
              (∀ i : nat, n ≤ i → i ≤ l → P i = true) ∧
              P (S l) = false}.

End Hypos.
